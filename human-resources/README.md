  # Human Resources Application

  Demo project for Human resource management with two entities: “Department” and “Employee”, related as one to many.
    
  ## Prerequisites
    
  jdk11  
  maven 3+  
  
  ## Installing  
  > mvn clean install  
  
  ## Running the tests  
  > mvn clean test
